# List icons font awesome

Generate json file who contains all font awesome icon value.

# Usage

```bash
$ node index.js
```

## Options

##### outFile

```bash
$ node index.js --outFile=myCustomFile.json
```

### Output

```json
{
  "version": "1",
  "icons": [
    "fa-500px",
    "fa-accessible-icon",
    "fa-accusoft",
    "fa-address-book",
    // ....
    "fa-youtube",
    "fa-youtube-square"
  ]
}
```
